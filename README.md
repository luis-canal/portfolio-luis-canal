# Portfólio Pessoal — Luis Canal

Este é meu portfólio pessoal desenvolvido com **HTML, CSS e JavaScript puro**, com foco em apresentar minha trajetória, formação, projetos e informações de contato de forma profissional, minimalista e responsiva.

O objetivo deste projeto é reunir minhas principais experiências, tecnologias e projetos em um único lugar, funcionando como minha apresentação profissional para recrutadores, professores e oportunidades futuras.

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts (Inter)

---

## Funcionalidades

* Layout moderno e minimalista
* Responsividade para desktop e mobile
* Menu mobile com botão hamburguer
* Alternância entre tema claro e escuro
* Persistência do tema com `localStorage`
* Hero section com foto de perfil e currículo em PDF
* Seção Sobre Mim com stack tecnológica
* Seção de Formação Acadêmica
* Seção de Projetos com links para GitHub
* Seção de Contato com links clicáveis
* Footer com copyright
* Animações suaves com `IntersectionObserver`
* Header com efeito ao scroll
